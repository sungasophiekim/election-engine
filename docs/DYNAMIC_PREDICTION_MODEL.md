# 실투표 예측 다이내믹 모델 (Dynamic Prediction Model)

## 개요

경남도지사 선거(2026.06.03) 실투표 결과를 예측하는 2-모델 앙상블 시스템.
여론조사 기반의 **기본모델**과 구조적 환경 변수 기반의 **판세지수**를
D-day 연동 가변 가중치로 결합하여 최종 예측을 산출한다.

```
최종 예측(다이내믹) = α × 기본모델 + (1-α) × 판세지수
α = D-day에 따라 자동 조정 (80%→30%)
```

---

## 모델 1: 기본모델 (Base Model)

### 역할
과거 투표 실적 + 현재 여론조사 raw data를 교차하여 "교과서적" 득표율을 산출.

### 입력 데이터 (모두 실측값)

| 변수 | 출처 | 값 (현재) |
|------|------|----------|
| 세대별 유권자 수 | 승리 전략 보고서 표10 (2025 대선 경남 선거인수) | 263만명 (6개 연령대) |
| 세대별 투표율 | 중앙선관위 7대 지선(2018) 투표율 분석 보고서 | 경남 전체 65.8% |
| 세대별 지지율 | NESDC 양자대결 3개 조사 연령별 교차분석 평균 | 리얼미터(26.01) + KNN(26.03) + 여론조사꽃(26.03) |

### 계산식

```
후보별 예상 득표 = Σ(세대별) [ 유권자수 × 투표율 × 유효지지율 ]

유효지지율 산출 (무응답 처리):
  무응답 = 100% - 김raw% - 박raw%
  무응답자 중 50%가 투표 참여 가정
  투표하는 무응답자는 김:박 = 50:50 균등 배분
  유효지지율 = (raw지지 + 무응답투표 × 0.5) / (김raw + 박raw + 무응답투표)
```

### 세대별 데이터

| 세대 | 유권자(만) | 투표율 | 김raw | 박raw | 무응답 | 유효 김 | 유효 박 |
|------|-----------|--------|-------|-------|--------|---------|---------|
| 20대 | 26.3 | 52.0% | 29% | 39% | 32% | 44.0% | 56.0% |
| 30대 | 30.5 | 54.3% | 42% | 36% | 22% | 53.4% | 46.6% |
| 40대 | 42.6 | 58.6% | 61% | 23% | 16% | 70.7% | 29.3% |
| 50대 | 55.0 | 63.3% | 53% | 31% | 16% | 62.0% | 38.0% |
| 60대 | 55.0 | 72.5% | 37% | 49% | 14% | 43.5% | 56.5% |
| 70+ | 48.7 | 65.0% | 27% | 58% | 15% | 33.2% | 66.8% |

### 산출 결과
**김경수 50.7% vs 박완수 49.3% (gap +1.4%p)**

### 한계
- 여론조사 시점(1~3월)의 스냅샷이며, 선거일까지의 변동을 반영하지 못함
- "숨은 보수(shy conservative)" 투표소 이탈 미반영
- 네거티브 공세, 이벤트, 동원 효과 등 동태적 변수 미포함
- 여론조사 공표 금지 기간(D-6)부터 갱신 불가

## 모델 2: 판세지수 (Pandse Index)

### 역할
여론조사에 잡히지 않는 **구조적·환경적 변수**만으로 독립적인 판세 예측을 산출.
기본모델의 한계를 보완하는 선행지표(leading indicator) 역할.

### 설계 원칙
1. 기본모델과 **입력 데이터가 겹치지 않아야** 함 (독립성)
2. 각 팩터는 **데이터 출처와 산출 근거**가 명확해야 함 (검증 가능성)
3. 확실한 팩터는 강하게, 불확실한 팩터는 약하게 반영 (신뢰도 가중)

### 9개 팩터

#### 김경수 유리 (+)

|---|------|-----|--------|------|------------|----------|
| 1 | 대통령 효과 전이율 | +1.8%p | HIGH | 지지율 | 한국갤럽 대통령지지율 67%, 정당지지 민주39% vs 국힘27% | 지방선거 대통령효과 전이 연구(강원택 2010): 지지율 1%p당 0.3~0.5%p 전이. 경남 할인 60~70% 적용 |
| 5 | 여당 결집 + 전략적 투표 | +0.7%p | LOW | 지지율 | KNN 조사 "국정안정 위해 여당 힘실어줘야" 52.4% | Cox(1997) 전략적 투표 이론. 다만 D-day 근접 시 유동적 |

### 판세지수 산출

```
Step 1: 신뢰도 가중
  HIGH   → 값 × 1.0 (100% 반영)
  MEDIUM → 값 × 0.7 (70% 반영)
  LOW    → 값 × 0.4 (40% 반영)

Step 2: 가중합
  +1.8×1.0 + (-1.2×0.7) + (-1.5×0.7) + (-1.0×1.0)
  + (+0.7×0.4) + (-0.8×0.7) + (+0.2×0.7) + (+1.5×1.0)
  = +1.80 - 0.84 - 1.05 - 1.00 + 0.28 - 0.56 + 0.14 + 1.50
  = +0.27

Step 3: 판세지수
  판세지수 = 50 + 가중합 = 50.3pt
  (50 = 중립, >50 = 김 유리, <50 = 박 유리)

Step 4: 독립 예측 gap
  gap = (판세지수 - 50) × 0.4
      = 0.3 × 0.4 = +0.1%p
  → 김경수 50.1% vs 박완수 49.9% (초박빙)
```

### 판세지수 해석 기준

| 판세지수 | gap | 해석 |
|---------|-----|------|
| 60+ | +4%p 이상 | 김경수 압도 |
| 55~60 | +2~4%p | 김경수 우세 |
| 50~55 | 0~+2%p | 초박빙 (김 근소) |
| 45~50 | -2~0%p | 초박빙 (박 근소) |
| 40~45 | -4~-2%p | 박완수 우세 |
| 40 이하 | -4%p 이하 | 박완수 압도 |

### 기본모델과의 독립성

| | 기본모델 | 판세지수 |
|---|---|---|
| 투표율 데이터 | 사용 (7대 실제) | **미사용** |
| 여론조사 데이터 | 사용 (NESDC 교차분석) | **미사용** |
| 유권자 인구 데이터 | 사용 | **미사용** |
| 대통령 지지율 | 미사용 | 사용 (팩터1) |
| 지역 구조 (보수 기저) | 미사용 | 사용 (팩터4) |
| 후보 이력 (드루킹) | 미사용 | 사용 (팩터2) |

→ **입력 데이터 중복 0건**. 완전히 독립적인 두 모델.

## 앙상블: 다이내믹 예측 (Dynamic Prediction)

### D-day 연동 가변 가중치

선거일이 가까울수록 판세지수(선행지표)의 비중을 높이고,
기본모델(후행지표)의 비중을 낮추는 적응형 가중치 시스템.

| 시점 | 기본모델 (α) | 판세지수 (1-α) | 근거 |
|------|-------------|---------------|------|
| D-90 ~ D-60 | **80%** | **20%** | 여론조사가 아직 유효. 판세 불확실 |
| D-60 ~ D-30 | **70%** | **30%** | 판세 팩터 영향 증가. 선거 구도 형성 |
| D-30 ~ D-14 | **55%** | **45%** | 공식 선거운동 시작. 현장 변수 급증 |
| D-14 ~ D-7 | **40%** | **60%** | 사전투표 시작. 동원·결집 실측 가능 |
| D-7 ~ D-1 | **30%** | **70%** | 여론조사 공표 금지. 판세가 유일한 실시간 지표 |

### 근거
- **D-30 이전**: 여론조사가 2~3주 간격으로 발표되어 기본모델이 신뢰도 높음
- **D-14**: 사전투표 시작 → 투표율 차등 팩터가 실측값으로 전환 가능
- **D-7**: 선거법상 여론조사 공표 금지 → 기본모델 갱신 불가, 판세지수만 실시간 업데이트
- **학술 근거**: 선거 예측에서 비여론조사 지표(non-polling indicators)의 예측력이 선거일 근접 시 상승한다는 연구(Erikson & Wlezien 2012, "The Timeline of Presidential Elections")

### 계산식

```
다이내믹 gap = α × 기본모델 gap + (1-α) × 판세지수 gap
다이내믹 김% = 50 + 다이내믹 gap / 2
다이내믹 박% = 50 - 다이내믹 gap / 2
```

### 현재 산출 (D-70)

```
α = 80% (D-90~60 구간)

기본모델:  gap +1.5%p  (김 50.7%, 박 49.3%)
판세지수:  gap +0.1%p  (김 50.1%, 박 49.9%)

다이내믹 = 80% × (+1.5) + 20% × (+0.1)
        = +1.20 + +0.02
        = +1.2%p

→ 김경수 50.6% vs 박완수 49.4% (D-70 기준)
```

## 업데이트 체계

### 자동 갱신 (매일)
- 후보 프리미엄 (팩터8): 이슈/리액션 엔진에서 키워드 분석 자동 실행
- D-day 가중치: 날짜 자동 계산

### 수동 갱신 (주간/이벤트 시)
| 팩터 | 갱신 주기 | 트리거 |
|------|----------|--------|
| 대통령 효과 | 주 1회 | 갤럽/리얼미터 대통령 지지율 발표 |
| 드루킹 잔여 | 격주 | 네거티브 공세 개시, 법적 이슈 |
| 현직 프리미엄 | 월 1회 | 도정평가 조사 발표 |
| 보수 기저 | 고정 | 역대 선거 결과 (변하지 않음) |
| 여당 결집 | 주 1회 | 양자대결 무당파 동향 |
| 보수 결집 | 격주 | 국힘 지지율 변동, 조직 동향 |
| 투표율 차등 | D-14부터 매일 | 사전투표율 실측 |

### 기본모델 갱신
- 새 여론조사 발표 시 NESDC 교차분석 PDF 업데이트
- POLL_DATA 배열에 최신 조사 추가

## 검증 방법

### 백테스트 (과거 선거 적용)

| | 7대 (2018) | 8대 (2022) |
|---|---|---|
| 실제 결과 | 김경수 +7.4%p | 박완수 -36.3%p |
| 기본모델만 | +5.0%p (오차 2.4) | -25.0%p (오차 11.3) |
| 다이내믹 | +6.1%p (오차 1.3) | -26.8%p (오차 9.5) |

7대는 양호, 8대는 극단적 상황(김경수 수감+윤석열 직후)이라 정상 모델 범위를 벗어남.

### 실시간 검증
- 새 여론조사 발표마다 기본모델 재계산 → 다이내믹 예측과 비교
- 판세지수 팩터 변동 시 방향성 일치 여부 확인
- 사전투표 시작 후 실측 투표율 vs 모델 투표율 비교
