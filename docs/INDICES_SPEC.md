# Election Engine v1 — 3대 지수 스펙

## 1. 이슈지수 (Issue Index)
> **"이슈가 얼마나 터졌는가"** — 감성이 아닌 순수 규모·속도 측정 (0~100점)

| 구분 | 내용 |
|------|------|
| **채널** | **7개** — 뉴스, 블로그, 카페, 유튜브, 구글트렌드, 커뮤니티(19+곳), 네이버데이터랩 |
| **1회 분석량** | 뉴스 ~330건(중복제거 후 100~200건) + 이슈키워드 10개 × 7채널 스캔 |
| **업데이트 주기** | **10분** |

### 구성요소

| 구성요소 | 배점 | 업데이트 주기 |
|----------|------|--------------|
| 뉴스 볼륨 | 25점 | 10분 (Naver News API) |
| 미디어 티어 | 20점 | 10분 (TV·포털·1티어 매체 분류) |
| 확산 속도 | 30점 | 10분 (6h/18h 속도비 + 이상치 + 전일비) |
| 후보 연결도 | 15점 | 10분 (직접언급·정책·지역 연관) |
| 채널 다양성 | 10점 | 10분 (5개+ 채널 동시 감지 시 만점) |

### 등급 기준
EXPLOSIVE(≥80) → HOT(≥60) → ACTIVE(≥40) → LOW(≥20) → DORMANT(<20)

## 2. 반응지수 (Reaction Index)
> **"사람들이 어떻게 반응하는가"** — 감성·참여·바이럴 측정 (0~100점)

| 구분 | 내용 |
|------|------|
| **채널** | **6개 레이어** — 커뮤니티(19+곳), 블로그/카페/유튜브, 뉴스+댓글, 구글트렌드 |
| **1회 분석량** | 이슈키워드 5~10개 × 19+커뮤니티 + 블로그·카페 각 10건 + 유튜브 검색결과 |
| **업데이트 주기** | **10분** |

### 구성요소

| 구성요소 | 배점 | 데이터 소스 | 업데이트 주기 |
|----------|------|------------|--------------|
| 커뮤니티 공명 | 25점 | 19+ 커뮤니티 바이럴 패턴 | 10분 |
| 콘텐츠 생성 | 20점 | 블로그·카페·유튜브 게시량 | 10분 |
| 감성 방향 | 20점 | 뉴스(35%)+블로그(25%)+카페(20%)+유튜브댓글(15%)+뉴스댓글(25%) 가중 | 10분 |
| 검색 반응 | 15점 | 구글트렌드 관심도·7일변화·급등감지 | 10분 |
| 유튜브 인게이지먼트 | 20점 | 댓글수·좋아요·조회수·댓글감성 | 10분 |

### 속도 보너스
- 이상치 + 서프라이즈 ≥80 → ×1.15
- 이상치 or 서프라이즈 ≥60 → ×1.10

### 등급 기준
VIRAL(≥75) → ENGAGED(≥50) → RIPPLE(≥25) → SILENT(<25)

## 3. 판세지수 (Pandse Index)
> **"숨겨진 유권자 이동을 추정"** — 이슈+반응+여론조사+경제·국정 종합 (0~100, 50=중립)

| 구분 | 내용 |
|------|------|
| **채널** | 이슈지수 + 반응지수 + 여론조사 + 국정지지율 + 경제지표 **통합** |
| **1회 분석량** | 상위 5개 이슈 + 전체 통합시그널 + 최신여론조사 + 국정·경제데이터 |
| **업데이트 주기** | **10분** (단, 여론조사·경제 등 원천데이터는 자체 주기) |

### 구성요소 (9 Factors)

| 구성요소 | 가중치 | 범위 | 원천 데이터 업데이트 |
|----------------------|--------|------|-------------------|
| 이슈 압력 | 15% | -50~+50 | 10분 (이슈지수 연동) |
| 이상치 시그널 | 10% | -50~+50 | 10분 (이상치 감지) |
| 반응 모멘텀 | 15% | -50~+50 | 10분 (반응지수 연동) |
| 소셜 속도 | 10% | -50~+50 | 10분 (구글트렌드+감성) |
| 여론조사 관성 | 20% | -50~+50 | 새 여론조사 발표 시 |
| 이슈지수 직접반영 | 12% | 0~12 | 10분 |
| 반응지수 직접반영 | 13% | 0~13 | 10분 |
| 대통령 효과 | 8% | -5~+8 | 주 1~2회 (국정지지율) |
| 경제 심리 | 5% | -3~+3 | 일/주간 (정부 통계) |
