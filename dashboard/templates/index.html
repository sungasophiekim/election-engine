승률 계산 방법
1. 최근 여론조사 3건을 선택합니다.
2. 가중치를 부여합니다:
가중치 = (1/경과일수 + 1) × √표본수
→ 최신 조사 + 대표본에 높은 가중치
3. 가중 평균 지지율을 계산합니다:
우리 평균 vs 상대 평균 → 격차(%p)
4. 오차범위를 결합합니다:
결합오차 = √(Σ(오차²×가중치) / 총가중치)
5. 정규분포로 승률을 추정합니다:
P(승리) = Φ(격차 / 결합오차)
→ 격차가 오차범위 대비 클수록 승률 ↑
⚠ 여론조사는 추정치입니다. 실제 투표 결과와 다를 수 있습니다.
